## About

`@discord-selfbot-sdk/util` is a collection of utility functions for use with @discord-selfbot-sdk/bot.

## Installation

**Node.js 22.12.0 or newer is required.**

```sh
npm install @discord-selfbot-sdk/util
yarn add @discord-selfbot-sdk/util
pnpm add @discord-selfbot-sdk/util
bun add @discord-selfbot-sdk/util
```
